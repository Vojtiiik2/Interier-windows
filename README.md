# Interier Windows – web v1

Statický vícestránkový web připravený pro GitHub Pages.

## Stránky
- index.html
- sluzby.html
- cenik.html
- realizace.html
- architekti.html
- kontakt.html

## Co upravit před ostrým spuštěním
1. Nahradit fotky v `assets/img`.
2. Ověřit orientační ceny.
3. Do formuláře v `kontakt.html` doplnit Make webhook do atributu `action`.
4. Doplnit reálné realizace a texty podle finální nabídky.
